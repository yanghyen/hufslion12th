### 8-9강 Django 개발 패턴 
장고 프레임워크의 패턴은 MVC 패턴  
Model
View
Controller -> Template 말고 Model, View 중점으로 진행  
Model: 장고가 ORM(Object Relational Mapping) 기술로 DB와 소통해줌  
View: req 받고 res 주는 처리하는 로직  
Template: ui 작업 (약간 front-end 영역)
